CATATAN GIT

Hai!

Kali ini saya akan menunjukkan pada kalian tentang GIT.

GIT adalah version control yang sifatnya lokal pada komputer kita. Fungsi git diantaranya adalah:

  • Mempermudah dalam develop aplikasi
  • Dapat mengetahui jejak perubahan dengan jelas
  • Apabila terjadi masalah kita punya "backup" dari perubahan sebelumnya

Perintah GIT yang sering dipakai:

git init

Untuk menginisialisasi suatu folder untuk dijadikan repositori dengan cara kita mengarahkan terminal tersebut ke dalam suatu direktori atau folder. Maka folder yang aktif itu akan menjadi suatu repositori

git add <nama file>

Kode tersebut akan mengarahkan file yang masih ada working directory ke dalam staging area. Jika kita ingin memasukan semua file tanpa terkecuali kita tinggal mengetikkan "git add ."
git status

Menampilkan semua file yang baru maupun yang sudah di modifikasi apakah sudah masuk ke staging area tau belum.
git commit -m "pesan"

Melakukan commit ke git dan perubahan akan dicatat oleh history.
